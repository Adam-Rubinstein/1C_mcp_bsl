When the user asks to install MCP, «установи mcp», or to set up the 1c-platform MCP server (optionally with a GitHub repository URL), follow the section «Установка MCP» in this file: https://raw.githubusercontent.com/Adam-Rubinstein/1C_mcp_bsl/main/.cursor/rules/use-1c-platform-mcp.mdc — use the Cursor workspace root folder as WS, and use the URL from the user message as the GitHub repo URL, defaulting to https://github.com/Adam-Rubinstein/1C_mcp_bsl if none was given. Run the described PowerShell or shell commands; do not ask for paths until a script fails.
